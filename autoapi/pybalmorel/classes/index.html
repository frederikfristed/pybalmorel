download_file(url, save_folder, filename=None)

Downloads a file from a given URL and saves it to a specified folder. Args:

url (str): The URL of the file to download. save_folder (str): The folder where the file should be saved. filename (str, optional): The name to save the file as. If not provided, the filename is extracted from the URL.

Returns:

str: The full path to the saved file.

Raises:

requests.exceptions.RequestException: If the download fails.

Notes:

  • The function ensures that the save folder exists.

  • The file is downloaded in chunks to handle large files efficiently.

chunk_size:

The size of each chunk of data to be written to the file. In this case, it is set to 8192 bytes (8 KB).
